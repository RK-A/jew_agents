USAGE INSTRUCTIONS
=============================================================================

Quick Start (5 minutes):
1. cp env.example .env && edit .env
2. docker-compose up -d
3. docker-compose exec backend python scripts/manage_data.py init
4. docker-compose exec backend python scripts/manage_data.py fill
5. docker-compose exec backend python scripts/manage_data.py sync
6. open http://localhost:8000/docs

Verify Installation:
python scripts/verify_installation.py

Test API:
./scripts/quick_api_test.sh

Check Status:
python scripts/manage_data.py status
